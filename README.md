# Webpack Boilerplate

A basic Webpack boilerplate with SASS, Typescript, prettier, dev server and eslint with airbnb guidelines.

## Installation

```
git clone https://github.com/EdvinasOak/Webpack-Boilerplate.git
npm i
```

## Usage

### Development server

```bash
npm run start:dev
```

### Production build

```bash
npm run build
```
